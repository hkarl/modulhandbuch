    Modulhandbuch
    Diese Webanwendung dient der Verwaltung von Modulhandbüchern über verschiedene Studiengänge hinweg. Lehrveranstaltungen, Module, etc. müssen nur einmal gepflegt werden. Aus den hier angelegten Daten können Modulhandbücher in verschiednen Formaten generiert werden, z.B. PDF, HTML, u.a.
    Sie haben folgende Möglichkeiten:

    Alle Daten ansehen oder ggf. ändern Handbücher generieren

    Aktualisierungen

    • 13.7.2015: Alle wesentlichen Elemente haben ein zusätzliches Feld "Editors": Hier kann der Besitzer eines Elements weiteren Nutzern Editierberechtigung geben. Bis auf Löschen ist ein Editor einem Besitzer gleichgestellt.
